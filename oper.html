JavaScript Operators

Let us take a simple expression 4 + 5 is equal to 9. Here 4 and 5 are called operands and '+' is called the operator. JavaScript supports the following types of operators:

  • Assignment Operators
  • Arithmetic Operators
  • Comparison Operators
  • Logical (or Relational) Operators
  • Conditional (or Ternary) Operator

Assignment Operators

Assignment operators are used to assign values to variables. Here's a list of commonly used assignment operators:


Operator Name Example
= Assignment operator a = 7; // 7
+= Addition assignment a += 5; // a = a + 5
-= Subtraction assignment a -= 2; // a = a - 2
*= Multiplication assignment a *= 3; // a = a * 3
/= Division assignment a /= 2; // a = a / 2
%= Remainder assignment a %= 2; // a = a % 2
**= Exponentiation assignment a **= 2; // a = a**2

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations. The following operators are known as JavaScript arithmetic operators:


Operator Name Example
+ Addition 10 + 20 = 30
- Subraction 20 - 10 = 10
* Multiplication 50*2 = 100
/ Division 30/5 = 6
% Modulus(Remainder) 20%10 = 0
++ Increment var a=10; a++; Now a= 11
-- Decrement var a=10; a--; Now a= 9

Comparison Operators

Comparison operators compare two values and return a boolean value, either true or false. The following operators are known as JavaScript comparison operators:


Operator Description Example
== Equal to: retruns true if the operands are equal. x == y
!= Not equal to: returns true if the operands are not equal x != y
=== Strict equal to: true if the operands are equal and of the same type x === y
!== Strict not equal: true if the operands are equal but of different type or not equal at all x !== y
> Greater than: true if left operand is greater than the right operand x > y
>= Greater than or equal: true if left operand is greater than or equal to the right operand x >= y
< Less than: true if the left operand is less than the right operand x < y
<= Less than or equal to: true if the left operand is less than or equal to the right operand x <= y

Example 2: Comparison operators in JavaScript

JavaScript comparison operators

        
               // equal operator
               console.log(2 == 2); // true
               console.log(2 == '2'); // true

               // not equal operator
               console.log(3 != 2); // true
               console.log('hello' != 'Hello'); // true

               // strict equal operator
               console.log(2 === 2); // true
               console.log(2 === '2'); // false

              // strict not equal operator
              console.log(2 !== '2'); // true
              console.log(2 !== 2); // false

Logical Operators

Logical operators perform logical operations and return a boolean value, either true or false. The following operators are known as JavaScript logical operators:


Operator Description Example
&& Logical AND: true if both the operands are true, else retruns false (10 == 20 && 20 == 30) = false
|| Logical OR: true if either of the operands is true; returns false if both are false (10 == 20 && 20 == 30) = false
! Logical NOT: true if the operand is false and vice-versa !(10 == 20) = true

Example 3: Logical operators in JavaScript

JavaScript logical operators

        
               // logical AND
               console.log(true && true); // true
               console.log(true && false); // false

               // logical OR
               console.log(true || false); // true

              // logical NOT
              console.log(!true); // false

Output:

                      true
                      false
                      true
                      false

Conditional (Ternary) Operator

JavaScript also contains a conditional operator that assigns a value to a variable based on some condition.

Syntax:

            
                variablename = (condition) ? value1:value2

Example 4: Conditional operator in JavaScript
In the example below, if the variable age is a value below 18, the value of the variable voteable will be "Too young", otherwise the value of voteable will be "Old enough".

JavaScript conditional operator

        
            let voteable = (age < 18) ? "Too young":"Old enough";
