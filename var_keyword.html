JavaScript Var

In JavaScript, the var keyword is used to declare a variable. Variables declared with var are function-scoped, meaning they are only accessible within the function in which they are declared. If a variable is declared outside of any function, it becomes a global variable.

Syntax:

            
                  var variableName = valueOfVar;

The variables declared using the var statement are hoisted at the top and are initialized before the execution of code with a default value of undefined. The variables declared in the global scope that is outside any function cannot be deleted.

Example 1: Basic Variable Declaration
Here, a variable named age is declared and assigned the value 25. The var keyword is used to declare the variable.

JavaScript var

            
                  var age = 25;

Example 2: Declaring multiple variables using the var keyword
In this example, we are declaring and initializing three variables: name, age, and isStudent using the var keyword. Each variable is assigned a different value.

JavaScript var

            
                  var name = 'John Doe',
                      age = 25,
                      isStudent = true;

Example 3: Variable Reassignment
In this example, the variable count is declared and initialized with the value 10. Later, its value is modified by adding 5 to it.

JavaScript var

            
                  var count = 10;
                  count = count + 5;

Example 4: Function Scope
In this example, the variable x is declared inside a function using the var keyword. It has function scope, meaning it is only accessible within the function.

JavaScript var

            
                  function example() {
                      var x = 5;
                      console.log(x);
                   }

Example 5: Global Scope
In the example below, the variable globalVar is declared outside any function, making it a global variable. It can be accessed from any part of the code, including functions.

JavaScript var

            
                  var globalVar = "I am global!";
                  function example() {
                  console.log(globalVar);
                  }

Example 6: Hoisting
Due to hoisting, the variable declaration is moved to the top of its scope during the compilation phase. However, the assignment remains in place. In the first console.log, message is declared but not yet assigned, so it logs undefined.

JavaScript var

            
                 console.log(message); // undefined
                 var message = "Hello, hoisting!";
                 console.log(message); // Hello, hoisting!
