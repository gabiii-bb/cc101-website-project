if...else Attributes

While writing a program, there may be a situation when you need to adopt one out of a given set of paths. In such cases, you need to use conditional statements that allow your program to make correct decisions and perform right actions.
JavaScript supports conditional statements which are used to perform different actions based on different conditions. Here we will explain the if..else statement.
Flow Chart of if-else adopt The following flow chart shows how the if-else statement works.


JavaScript supports the following forms of if..else statement:
  • if statement
  • if...else statement
  • if...else if... statement

Example 1: if statement

The given JavaScript code initializes an age variable with a value of 20 and checks if it is greater than 18 using a conditional statement " if (age > 18)". If the condition is true, the code dynamically writes "Qualifies for driving." in bold to the document using the document.write method.

              <html>
                 <body>
                   <script type="text/javascript">
                      <!--
                          var age = 20;
                           if (age >18){
                              document write("<b>Qualifies for driving.</b>");
                           }
                        //-->
                   </script>

                 </body>
               </html>

Example 2: if-else statement

Unlike the first example, the code shown below no longer meets the condition statement if (age < 18)" so the code will not execute the contents of its document.write. Instead, it will proceed to the else block and writes "Does not qualify for driving". using the same document.write

Example 3: if-else if

In this example, the code assigns the value to the variable name 'book'. The code first check if book is equal to history and, if true, writes "History book" in bold to the document. If the initial condition is false, it proceeds to the next 'else if' statement, checking if book is equal to "maths". If this condition is true, it writes "Maths Book" to the document, and this process will continue until the condition are meet. However, if none of the conditions are met, it writes "Unknown Books". 

    
        <!DOCTYPE html>
           <html>
              <body>
                 <script type="text/javascript">
                    <!--
                    var book = "maths";  

                    if (book == "history" ) {
                          document.write("<b>History Book</b>");
                       } else if (book == "maths" ) {
                          document.write("<b>Maths Book</b>");
                       } else if (book == "economics" ) {
                          document.write("<b>Economics Book</b>");
                       } else {
                          document.write("<b>Unknown Book</b>");
                       }
                   //-->
                 </script>
                 <p>Set the variable to different value and then try...</p>
 
              </body>
           </html>

Output:

        Maths Book
