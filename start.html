Introduction to JavaScript

What is JavaScript?

JavaScript is a widely used programming language that enables interactive and dynamic content on websites. It is primarily known for client-side scripting,allowing developers to create responsive and engaging user interfaces. JavaScript is supported by most web browsers and can be integrated with HTML and CSS to build interactive web pages.

History of JavaScript

JavaScript was created by Brendan Eich in 1995 while he was working at Netscape Communications Corporation. Originally called "Mocha" and later "LiveScript," it was eventually renamed JavaScript when Netscape formed a partnership with Sun Microsystems.

In 1997, Netscape submitted JavaScript to the Ecma International standards organization, leading to the development of the ECMAScript standard. ECMAScript is the standardized specification upon which JavaScript implementations are based.

Over the years, JavaScript evolved, gaining popularity for its ability to add interactivity to web pages. Major advancements, such as the introduction of AJAX (Asynchronous JavaScript and XML), contributed to its role in creating more dynamic and responsive web applications.

In the 2010s, with the rise of single-page applications and the availability of powerful JavaScript frameworks like Angular, React, and Vue.js, JavaScript became even more integral to web development, expanding its use beyond the browser to server-side development with the advent of Node.js. Today, JavaScript is a fundamental technology for web development, powering a vast array of websites and applications

Java vs. JavaScript

Java and JavaScript are distinct programming languages serving different purposes. Java, designed for platform-independent applications, is utilized in server-side development, Android mobile apps, and large-scale enterprise systems. It is an OOP (Object-Oriented Programming) language and it relies on a Java Virtual Machine (JVM) for execution, compiling code into bytecode interpreted by the JVM. On the other hand, JavaScript is predominantly employed for client-side web development, enhancing web page interactivity. It executes directly in web browsers without the need for a separate runtime environment. While both languages share some syntax similarities, Java is a class-based, statically-typed language, while JavaScript is dynamically-typed and follows prototype-based object orientation. Despite their names, Java and JavaScript are used in distinct application domains and environments.

The <script> tag

This element in HTML is used to embed or reference JavaScript code within an HTML document. It allows you to include JavaScript directly in your HTML file or link to an external JavaScript file.

Script Tag Attributes

            
            <script language="javascript" type="text/javascript">
                JavaScript code
            </script>

Script Tag Attributes

            
            <script language="javascript" type="text/javascript">
                JavaScript code
            </script>

Attributes:

  • language: Specifies the scripting language (deprecated in modern HTML).
  • type: Indicates the scripting language; recommended value is "text/javascript".

Hello World Using JavaScript

Here is an example of JavaScript code that outputs "Hello World!".

JavaScript Syntax

            
            <html>
                <body>
                    <script language="javascript" type="text/javascript">
                        
                                document.write("Hello World!")
                       
                    </script>
                </body>
            </html>

JavaScript Syntax:

  • document.write: Writes a string into the HTML document.

Output:

Hello World!
