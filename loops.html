JavaScript Loops

While writing a program, you may encounter a situation where you need to perform an action over and over again. In such situation, you would need to write loop statements choose the number of lines.

The while Loop

The most basic loop in JavaScript is the while loop which would be discussed in this chapter. The purpose of a while loop is to execute statement or code block repeatedly as long as an expression is true. Once the expression becomes false, the loop terminates.

Syntax
The syntax of while loop in JavaScript is as follows: 

            
            while (expression){
               Statement(s) to be executed if expression is true
            }

Example: While loop

The JavaScript code initializes a variable named 'count' to 0 and uses a while loop to iteratively write the current count value to the document. The loop continues until the count is less than 10, incrementing the count by 1 in each iteration. After the loop concludes, the script writes "Loop Stopped!" to the document.

              //-->

The do...while Loop

The do...while loop is similar to the while loop except that the condition check happens at the end of the loop. This means that the look will always be executed at least once, even if the condition is false.

Flow Chart
The Flow Chart of do-while loop looks as follows:



Syntax
The syntax of do-while loop in JavaScript is as follows: 

        
            do (expression){
                Statement(s) to be executed if expression is true
            } while (expression);

Example: do-while loop

The JavaScript code initializes a variable named 'count' with a value of 0, then executes a do-while loop to check the loop condition. The script dynamically writes the current count value to the document, concatenating it with "Current count:". The loop continues as long as count is less than 5, and the script writes "Loop Stopped!" after completion.

                        
        
<!DOCTYPE html>
    <html>
        <body>
            <script type="text/javascript">
              <!--
                var count = 0;
                    document.write("Starting loop" + "</br>");
   
                do {
                    document.write("Current count: " + count + "<br>");
                }
                while ( count < 5 );
                document.write("Loop Stopped!");
   
              //-->
            </script>
            <p>Set the variable to different value and then try...  </p>
        </body>
    </html>

Output:

            
            Starting Loop
            Current Count : 0
            Current Count : 1
            Current Count : 2

JavaScript - For Loop

The 'for' loop is the most compact form of looping. It includes the following three important parts:

  • The loop initialization where we initialize our counter to a starting value. The initialization statement is executed before the loop begins.
  • The test statement which will test if the given condition is true or not. If the condition is true, then the code given inside the loop will be executed, otherwise the control will come out of the loop.
  • The iteration statement where you can increase or decrease your counter.

You put all the three parts in a single line seperated by semicolons.

Syntax

            } while (expression);

Example: for loop

This JavaScript code snippet declares a variable named 'count' and uses a for loop to iterate until the count is less than 10. The script dynamically writes the current count value to the document, concatenating it with the string "Current count:", with each iteration followed by a line break for better readability. Once the loop completes, it writes "Loop Stopped!" to the document.

        
<!DOCTYPE html>
    <html>
        <body>
            <script type="text/javascript">
              <!--
                var count;
                document.write("Starting loop" + "</br>");

                for (count = 0; count <10; count++){
                    document.write("Current count: " + count );
                    document.write("</br>");
                }
                document.write("Loop Stopped!");
              //-->
            </script>

        </body>
    </html>
